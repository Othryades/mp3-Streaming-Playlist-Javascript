# MailCatcher

Mp3 Streamer

## Features

* Save mp3 steam URL.
* Can save few url under same playlist.
* save playlist in local storage.


## How

1. Run in Localhost (Uwamp, Xampp)
2. Download all the files in localohost folder
3. Run!
